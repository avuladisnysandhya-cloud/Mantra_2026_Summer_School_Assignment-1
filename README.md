This repository contains my submission for **Assignment 1: Frontend Website Design, SEO, Responsiveness & Netlify Hosting** as part of the MANTRA 2026 Summer School program. 

The project focuses on creating a premium, fully responsive luxury resort website integrated with modern frontend practices, search engine optimization (SEO) basics, and a live AI conversational assistant.

## 🛠️ Core Features Implemented

### 1. Frontend Website Design
- Built a clean, modern, and luxury-themed user interface using HTML5 and CSS3.
- Designed a custom color palette utilizing a dark Navy Blue and White layout to give a premium hospitality feel.

### 2. Mobile Responsiveness
- Implemented responsive web design principles (using CSS Media Queries / Flexbox / Grid).
- Ensured the layout adapts flawlessly across various devices, including desktops, tablets, and smartphones.

### 3. Search Engine Optimization (SEO)
- Added structured Meta Tags (`title`, `description`, `keywords`) in the HTML header for better indexability.
- Used semantic HTML tags (`<header>`, `<main>`, `<section>`, `<footer>`) to optimize page structure.
- Ensured all images contain proper `alt` text for web accessibility and image SEO.

### 4. AI Chatbot Integration (Mirage)
- Successfully integrated a live AI agent named **Mirage** via tawk.to.
- Custom-trained the AI core with rules to handle resort-specific inquiries (such as room rates starting at Rs. 12,000/night, amenities like the infinity pool/spa, and booking workflows).
- Tailored the chatbot widget UI with a matching Navy Blue and White theme.

### 5. Netlify Hosting
- Connected the project repository with Netlify for continuous deployment and live hosting.
